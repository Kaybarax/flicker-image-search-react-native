## React Native Flickr Image Search App


### Dev
`$ git clone <repo>`

`$ yarn install`

`$ yarn start`


#### Android
`$ npx react-native run-android`


#### iOS
`$ npx pod-install ios`

`$ npx react-native run-ios`


### Release
#### Test Release Android
`$ cd android && ./gradlew installRelease`


#### Build Release Android
`$ cd android && ./gradlew assembleRelease`


#### Test and Build Release iOS
`Use XCode and TestFlight`
